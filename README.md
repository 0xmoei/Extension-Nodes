### Monitor VPS Recources (if you are running a Linux VPS)
**Install htop**
```console
sudo apt install htop
```

**Open htop**
```
htop
```
* Now you can check the usage of CPU cores, Memory(RAM), swap RAM, and manage the live tasks on your VPS
